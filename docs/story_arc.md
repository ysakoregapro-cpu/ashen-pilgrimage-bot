# 灰星巡礼録 — ストーリーアーク

## 物語の核

世界は一度、壊れかけた。だが滅びなかった。誰かが繋ぎ止め、折れた道を結び直し、消えかけた灯を守った。

救われた世界は元通りにはならない。灰の下に星が残り、記録されなかった頁が残り、選ばれなかった時間が残る。

プレイヤーは世界を救う勇者ではなく、**救われたあとに何が残されたかを確かめる巡礼者**である。最後に、**次の繋ぎ手**になる。

## 繋ぎ手とは

かつて世界の断絶を繋ごうとした者たち。アオイ、ユイ、カイ、レン、シズク、ジン、クラトがその名残を担う。

序盤では「事情を知る案内人」として登場。中盤で「繋ぎ手」の言葉が現れ、ヴァルハラ踏破後にアオイが中心にいたことが明かされる。

## 各町の役割

- **星原**: 旅の始まり。「席につきたまえ」の最初の意味
- **薄明の港**: 灯火と救難、帰る場所
- **白銀鉱山**: 武器と打ち直し、Srcの前兆
- **霧の森**: 守ること、守れなかった名
- **月下図書館**: 記録、旧王、ヴァルハラの名
- **地下市**: 代償と綺麗ではない選択
- **砂時計の都**: 時間と繋ぎ手の過去
- **深層炉**: 世界改変の残響
- **ヴァルハラ**: 最後の巡礼
- **星原（終章）**: 次の繋ぎ手への継承

## ヴァルハラまでの流れ

1. 星原で巡礼開始 → 初勝利 → 星原ボス
2. 薄明〜砂時計まで順に町解放・章ボス
3. 深層炉で星印8つ → ヴァルハラ解放
4. 旧王の残響撃破 → 終章

## 終章「繋ぎ手」

ヴァルハラ踏破後、星原の古びた円卓へ。アオイが再び言う。

> 席につきたまえ。次は、君が平和な世を繋ぐ番だ。

始まりと同じ言葉が、まったく違う意味で響く。
